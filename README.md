# lab1-timp

Лабораторная работа №1 ТИМП, НГТУ ЗИ.
Вариант 19 — «Безопасность в медицинских учреждениях».

Журнал инцидентов безопасности, связанных с обращением лекарственных средств в медицинских учреждениях: SPA на React с REST API на json-server. CRUD, клиентский роутинг, валидация форм, обработка ошибок, страница 404, статистика и фильтры, сортировка по колонкам. Персональные данные обезличены методом введения идентификаторов (Приказ Роскомнадзора № 996, ст. 3 п. 9 ФЗ-152, ст. 13 ФЗ-323).

Развёрнутая версия: https://lab3.t-queue.ru/

## Маршруты

- `/` — список инцидентов
- `/detail/:id` — карточка инцидента
- `/add` — форма добавления
- `/edit/:id` — форма редактирования
- `*` — 404

## Локальный запуск

```bash
npm install
npm run server   # json-server, порт 5050
npm run dev      # vite,        порт 5173
```

Открыть http://localhost:5173.
